# Problem

Given a floating-point image:

```cpp
float* input;
float* output;

width;
height;
```

Process the image using 128Г—32 tiles.

For each tile:

1. Compute the minimum value inside the tile.
2. Compute a 16Г—16 stencil-based transformation.
3. Normalize stencil results by the tile minimum.

Reference operation:

```cpp
tile_min =
   minimum value inside 128Г—32 tile;

output[y][x] =
   stencil16x16(input, x, y) /
   max(tile_min, 1e-6f);
```

---

# Stencil Definition

```cpp
float acc = 0.0f;

for (int dy = -7; dy <= 8; ++dy)
{
   for (int dx = -7; dx <= 8; ++dx)
   {
       float v = input[y + dy][x + dx];

       float transformed =
           v * v +
           0.25f * v +
           sqrtf(fabsf(v));

       acc += coeffs[dy + 7][dx + 7] * transformed;
   }
}
```

Result:

```cpp
output[y][x] =
   acc / max(tile_min, 1e-6f);
```
